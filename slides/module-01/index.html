Read

Use company data.

Think

Choose a next step.

Security layer 3 · Microsoft Defender

Watch what agents do while they run.

Audit

Record risky behavior.

Detect

Find unsafe or malicious actions.

Block

Stop supported risky actions.

Start by watching. Block only high-confidence threats.

Admin shows · Threat security

Then: check runtime protection.

security.microsoft.com→Settings→Security for AI→Policies & rules→Real-time protection

Default rule

Audits agent activity.

Custom rule

Can block selected threats.

Treat prompt evidence as sensitive data.
